## Rationale
- **Root cause (L067–L074):** `match` is registered on `posedge clk`, so it reflects `pstate` from the *previous* cycle. The reference drives `match` combinationally from the current `pstate` (see `RefModule` `always @(pstate or rst_n)`).
- **Waveform timing:** Every detection pulse on `match_ref` is **10 ps earlier** than `match_dut` (e.g. ref@75 vs dut@85, ref@185 vs dut@195) — exactly one half-period of the 10 ps clock, consistent with a one-cycle output delay.
- **First mismatch @ 5 ps:** At t=0, `match_dut` is `X` (uninitialized `reg`) while `match_ref` is `0` during reset; the causal trace shows `tb.match_dut @ t=0: x` with `rst_n=0`.
- **65 / 1044 mismatches:** All are on `match`; the FSM next-state logic (L026–L065) already matches the reference — no state-transition changes were required.
- **Spec alignment:** After the fourth valid bit completes `0110`, `pstate` becomes `s4_d0110` in the same timestep the combinational `nstate` logic evaluates; `match` must go high in that same timestep, not on the following clock edge.
- **Fix applied:** Replace the clocked `match` block with combinational output `always @(pstate or rst_n)`, asserting `match=1` when `!rst_n` is inactive and `pstate==s4_d0110`. Simulation passes with **0 mismatches**.

## FixedTopModule
```verilog
module TopModule(
	input clk,
	input rst_n,
	input data,
	input data_valid,
	output reg match
	);


reg [3:0] pstate,nstate;

parameter idle=4'd0,
		  s1_d0=4'd1,
          s2_d01=4'd2,
          s3_d011=4'd3,
          s4_d0110=4'd4;

always @(posedge clk or negedge rst_n)
begin
    if(!rst_n)
        pstate<=idle;
    else
        pstate<=nstate;
end

always @(pstate or data or data_valid)
begin
    case(pstate)
        idle:
            if(data_valid && !data)
                nstate=s1_d0;			//First bit matched
            else
                nstate=idle;
        s1_d0:
            if (data_valid)
				begin	
					if (data) nstate = s2_d01;		//Data valid and is 1, first two bits 01 matched, next state is s2_d01
					else nstate = s1_d0;			//Data valid but is 0, only first bit 0 matched, next state is s1_d0
				end
			else nstate = s1_d0;					//Data invalid, stay in s1_d0
        s2_d01:
            if (data_valid)
				begin	
					if (data) nstate = s3_d011;		//Data valid and is 1, first three bits 011 matched, next state is s3_d011
					else nstate = s1_d0;			//Data valid but is 0, only first bit 0 matched, next state is s1_d0
				end
			else nstate = s2_d01;					//Data invalid, stay in s2_d01
        s3_d011:
            if (data_valid)
				begin	
					if (!data) nstate = s4_d0110;		//Data valid and is 0, first four bits 0110 matched, next state is s4_d0110
					else nstate = idle;					//Data valid but is 1, no match, next state is idle
				end
			else nstate = s3_d011;					//Data invalid, stay in s3_d011
        s4_d0110:
            if (data_valid)
				begin	
					if (!data) nstate = s1_d0;		//Data valid and is 0, matches first bit 0 of target sequence, next state is s1_d0
					else nstate = idle;			//Data valid but is 1, does not match target sequence, next state is idle
				end
			else nstate = idle;					//Data invalid, next state is idle
        default:
            nstate=idle;
        endcase
end

always @(pstate or rst_n)
begin
    if (!rst_n)
        match = 1'b0;
    else if (pstate == s4_d0110)   // All four bits matched
        match = 1'b1;
    else
        match = 1'b0;
end

endmodule
```
